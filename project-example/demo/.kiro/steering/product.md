# Product Summary

Summer Reading Tracker — a web application that allows users to register an account and track their reading over the summer. Users record books they've read, including title, author, genre, and page count.

## Core Domain

- **Users**: Registration and authentication (JWT-based)
- **Books**: CRUD operations for a user's reading log (title, author, genre, page count)

## Key Goals

- Simple REST API for user and book management
- Persistent storage via SQLite
- Stateless authentication using JSON Web Tokens
